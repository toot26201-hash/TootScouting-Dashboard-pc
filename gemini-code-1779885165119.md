# TootScouting Performance Dashboard

أداة تحليل أداء احترافية مصممة لتحليل بيانات المباريات لفرق كرة القدم. 
تعتمد الأداة على معالجة إحداثيات البيانات (Event Data) لاستخراج رؤى تكتيكية.

## المميزات:
- رسم خرائط حرارية (Heatmaps) للتحركات والضغط.
- تحليل إحصائي للأداء الفردي والجماعي.
- مقارنة الأداء (Percentiles) بناءً على إحصائيات الدوري.

## التقنيات المستخدمة:
- Streamlit (لواجهة المستخدم)
- Pandas (لمعالجة البيانات)
- mplsoccer (لتحليل البيانات التكتيكية)
